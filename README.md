#### ☕ Bem-vindo ao meu Java Lab! 🧪</br>
Sim, eu sei, Java é "verboso".</br>
Mas, aqui a gente transforma essa verbosidade em projetos organizados, tipagem forte e POO de respeito! 💪

A jornada para dominar a JVM é longa, mas, o café é forte! 💪☕</br>
Este repositório centraliza meus avanços e tropeços no mundo Java.
